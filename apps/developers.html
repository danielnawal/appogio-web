Funcionalidades

Construye sobre la plataforma GPS

Documentación interactiva de la API

Documentación clara y navegable de cada endpoint, con autenticación, parámetros y ejemplos de respuesta listos para copiar.

Claves de acceso por proyecto

Genera y revoca API keys por proyecto, con control de quién accede a qué y trazabilidad de cada llamada.

Más de 15 endpoints REST

Accede a vehículos, posiciones, eventos, geocercas y más con una API REST consistente y predecible.

Autenticación Bearer y respuestas JSON

Estándar moderno: autenticación con token Bearer, respuestas en JSON y rate limiting documentado.

Ejemplos de código y Postman

Ejemplos en cURL, Python, JavaScript y PHP, más una colección Postman y especificación OpenAPI para arrancar rápido.

Sandbox para probar en vivo

Prueba las llamadas a la API directamente desde el portal, sin escribir una línea de código, antes de integrar.
